## Installation

The latest precompiled binaries are available on [github](https://github.com/nix-community/nurl/releases/latest).

Alternatively you can install nurl from [crates.io][crate] with cargo.

```sh
cargo install nurl
```


## Building from source

```sh
cargo build --release
```
